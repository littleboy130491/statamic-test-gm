Usage

How to Use

Quick guide to customizing the design system.

1. Modify settings.css

Change color tokens, typography mappings, and spacing values to match your project's design language.

2. Extend with project.css

Add project-specific classes, component variants, and custom styles.

3. Use Utility Classes

Apply .container, .flow, .stack, .cluster, and .grid for consistent layouts.

4. Reference Tokens

Use CSS custom properties like var(--space-4) and var(--color-primary) in your custom styles.
